My projects

bizstore.si

  • implementing technologies:

    Application runs on AWS EC2 instance and is deployed with fabric into development, integration and production environment. It’s build within Django framework and for data management we used third party application south, which is almost a standard for data migration in Django.

  • overview:

    Application is meant for telecom company which recently entered cloud brokerage market, to offer potential client easy to access information about services they offer. Visiting party may also order one or more services online right through the application and never need to stop by the telecoms shop. App is made as a CMS, where templates are logically chosen according the url entry client sent.

  • features:

    Simple login with made with django forms (crispy forms) to enter user profile, which is linked to django's user.auth module Social login implemented with the third party component (janrain), enabling easy access. List of services/products according to category they come from Automatically generated price list according to the products price components from database. Product include features that are orderable or come in bundle with the product. Shopping cart check out / order procedure Revision of ordered services/product offer client was implemented for the purpose of enabling client to manage number of subscriptions online Messaging system is clients tool to communicate with technical and sales staff was implemented and connected to telecoms CRM

  • My role:

    I was involved in the project from the start, as a developer and sometimes GUI expert. From implementing logic which presented shopping cart and processes that supported messaging system, my task was also to keep design response, since our client was telecom company. At times this task was not easy, bcs of tabular data which had to be presented in a price list of products/services.

url: bizstore.si
